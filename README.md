📎 فاحص الروابط - Link Inspector

أداة ويب متكاملة لفحص الروابط وتحليلها باستخدام خدمات أمنية متعددة، مع واجهة مستخدم جذابة وتجربة تفاعلية سلسة.

✨ المميزات

· فحص الروابط بأربع خدمات مختلفة:
  · 🔍 URLQuery - تحليل شامل لسلوك الرابط والاتصالات
  · 🛰️ HATHR - منصة محلية للفحص والإبلاغ عن الروابط
  · 🌐 URLVoid - التحقق من سمعة النطاق عبر قواعد البيانات
  · 🛡️ VirusTotal - فحص عبر محركات الحماية والذكاء الاصطناعي
· ثلاث طرق لعرض النتائج:
  · عرض داخلي داخل الموقع
  · عرض منبثق (Popup)
  · فتح في نافذة جديدة
· واجهة مستخدم متطورة:
  · وضعين للثيم (فاتح/داكن) مع انتقال سلس
  · أيقونات متحركة في الخلفية
  · تأثيرات ثلاثية الأبعاد (3D) على الأزرار
  · شاشة تحميل أثناء الفحص

🚀 طريقة الاستخدام

1. فتح الملف - قم بفتح ملف index.html في المتصفح
2. إدخال الرابط - الصق الرابط المطلوب فحصه في حقل الإدخال
3. اختيار طريقة العرض - حدد الطريقة المناسبة لعرض النتائج
4. اختيار الخدمة - اضغط على أحد الأزرار الأربعة لبدء الفحص

🛠️ التقنيات المستخدمة

· HTML5 - هيكل الصفحة
· CSS3 - التصميم والتحريك (Animations, Transitions, 3D Transformations)
· JavaScript - المنطق والتفاعل
· Responsive Design - تصميم متجاوب مع جميع الشاشات
· Web APIs - استخدام iframe و Audio API

📦 التثبيت والتشغيل

```bash
# نسخ المستودع
git clone https://github.com/yourusername/link-inspector.git

# الدخول إلى المجلد
cd link-inspector

# فتح الملف في المتصفح
open index.html
# أو استخدم أي خادم محلي مثل Live Server
```

🎨 تخصيص الألوان

يمكنك تخصيص ألوان الخدمات بسهولة من خلال متغيرات CSS:

```css
:root {
  --urlquery-color: #4f46e5;
  --hathr-color: #16a34a;
  --urlvoid-color: #0284c7;
  --virustotal-color: #1d4ed8;
}
```
